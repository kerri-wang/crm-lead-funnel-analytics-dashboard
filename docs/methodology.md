# Methodology

## Project Goal
This project analyzes dealer performance, campaign efficiency, and conversion funnel health using synthetic CRM and sales data.

## Workflow
1. Create or load synthetic raw datasets for leads, campaigns, dealers, interactions, models, and outcomes.
2. Clean and standardize key fields such as dates, IDs, campaign labels, and dealer attributes.
3. Build curated tables for dashboard-ready analysis.
4. Define KPIs for conversion, qualification, follow-up behavior, campaign efficiency, and ROI.
5. Visualize results in Tableau across three views:
   - Dealer Comparison
   - Campaign Performance
   - Conversion Funnel View

## Analytical Focus
The dashboard is designed to help answer:
- Which dealers underperform despite similar lead volume?
- Which campaigns are efficient versus inefficient?
- Where do the largest funnel drop-offs occur?

## Notes
All data in this project is synthetic and created for demonstration purposes only.
